# Frontend Skeleton

Standalone Nuxt 4 + Vue 3 + TypeScript frontend scaffold with Pinia, Tailwind CSS, route middleware, reusable UI primitives, composables, and placeholder business-app pages.

## Quick start

```bash
npm install
cp .env.example .env
npm run dev
```

## Notes

- This project intentionally uses placeholder data and TODO markers.
- Forms are scaffolded with `vee-validate` and `zod`.
- API access is centralized through reusable composables and does not assume a live backend.
